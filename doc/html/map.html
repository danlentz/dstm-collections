keymap   f map  [Function]

return a newly created map where the keys of 'map' are replaced by the result of applying the function f to each key. Note that unless f is a properly 'invertable' function, the resulting map will not be very useful. This is included in the external interface mainly because it is extremely useful for manipulating SEQ type collections, where we rely on it heavily
