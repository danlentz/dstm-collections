do-orelse   &rest fns  [Function]

Perform one of the functions in the list fns. The list is examined in order, front to back. The first one to succeed is the sub-transaction accepted. If none succeed, or the overall transaction fails, the the whole thing is restarted.

execute-transaction   place fn  [Function]

safe-value   thing  [Function]

If thing is a DSTM variable, read and return its value within an atomic transaction.

If thing is not a DSTM variable, simply return its value.
